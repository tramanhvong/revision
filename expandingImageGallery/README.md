# Design an expanding image gallery template
- Image expands when the user hvoers over the image.
- Intuitive expansion of images.

## Approach
- Basic structure of webpage using div and paragraph eleemnts.
- Link external stylesheet to html. id1, id2, id3 defines different background images.
- Attribute class img_div contains various properties including height, width, border radius.
- Property transition is used for smooth expanding effects.
